## Introduction

技术栈：

*   webpack
*   antd
*   React
*   React-Router-Dom
*   fetch(whatwg-fetch)

特性：

*   可使用动态 import 或 bundle-loader(require.ensure)进行页面异步加载
*   使用 react-hot-loader 实现热加载
*   请求模块封装 fetch，并支持添加请求前后的回调事件。

## Usage

development:

```
npm install

npm start
```

production:

```
npm run build
```
